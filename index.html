Noisy Search[beta]

Noisy Search allows you to submit a word or phrase to be searched for within a randomly generated string of characters.

How To Use:

Enter a word for the program to attempt to generate randomly. Keep in mind, each character is individually randomly generated. So, the longer the word, the lower the probability it will be generated. Thus, taking a longer time to generate.
